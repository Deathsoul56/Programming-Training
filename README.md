# Programming-Training

Este repositorio contiene una colección de programas y ejercicios desarrollados para aprender y practicar varios lenguajes de programación. El objetivo es explorar conceptos fundamentales y avanzados en diferentes entornos y sintaxis.

## Estructura del Proyecto

- **C-C++/**: Ejercicios y ejemplos en C y C++.
- **Go/**: Programas y scripts en Go.
- **Kotlin/**: Ejercicios en Kotlin.
- **MatLab/**: Scripts y ejemplos en MATLAB.
- **MongoDB/**: Scripts y ejemplos de consultas y manejo de bases de datos MongoDB.
- **Python/**: Ejercicios y scripts en Python.
- **R/**: Ejercicios en R.
- **Ruby/**: Ejercicios en Ruby.
- **Swift/**: Ejercicios en Swift.

## ¿Qué encontrarás aquí?

- Programas de introduccion a los distintos lenguajes.
- Ejercicios de condiciones, ciclos, funciones, arreglos y recursividad.
- Ejemplos de manejo de archivos, bases de datos y operaciones matemáticas.
- Scripts para practicar conceptos básicos y avanzados de cada lenguaje.
